# Java-Typing-Test
## About
A simple console application built to practice string manipulation and time tracking in Java. The program gives the user 15 random programming words to type and calculates their Words Per Minute (WPM) based on how fast they finish. Time is measured using `System.currentTimeMillis()`.

## Features
* Generates 15 random words for each test
* Calculates typing speed (WPM)
* Shows accuracy percentage (correct vs. total words)
* Case-insensitive checking (ignores uppercase/lowercase)
